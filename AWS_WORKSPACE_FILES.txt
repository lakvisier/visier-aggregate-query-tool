FILES TO COPY TO AWS WORKSPACE
================================

Copy these files to your AWS workspace (all in the same directory):

REQUIRED FILES (minimum):
1. test_impersonation.py
2. client.py
3. requirements.txt

OPTIONAL (for reference/documentation):
- impersonator.md
- API_REFERENCE.md

SETUP IN AWS WORKSPACE:
1. Copy the 3 required files above
2. Create .env file with your credentials (see below)
3. Install dependencies: pip install -r requirements.txt
4. Run: python3 test_impersonation.py --test-token-only

.env FILE TEMPLATE:
-------------------
VISIER_IMPERSONATION_MY_USERNAME=your_username
VISIER_IMPERSONATION_MY_PASSWORD=your_password
VISIER_IMPERSONATION_JURISDICTION=US
VISIER_IMPERSONATION_TARGET_USERNAME=customer.user@example.com
VISIER_IMPERSONATION_TARGET_TENANT_ID=customer-tenant-123
VISIER_IMPERSONATION_CUSTOMER_APIKEY=customer_api_key

QUICK COMMANDS:
---------------
# Test step 1 (secure token)
python3 test_impersonation.py --test-token-only

# Test step 2 (impersonation token)
python3 test_impersonation.py --test-impersonation-token
